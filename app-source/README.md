# app-source

This folder contains the editable source for the BROAGENTS interface.

- `src/` contains the React dashboard source.
- `main.js` and `preload.js` are the Electron shell files from the current project.
- `index.html` and `vite.config.js` are included so the UI can be rebuilt later.

For normal use you do not need this folder. The normal runtime already uses `dist/`.
