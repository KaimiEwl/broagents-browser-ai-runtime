# Как пользоваться BROAGENTS

## Быстрый старт

1. Положите папку `BROAGENTS` в корень проекта.
2. Запустите `start-broagents.cmd`.
3. Если нужен desktop app, запустите `start-broagents-full.cmd`.
4. В Chrome откройте `chrome://extensions/` и загрузите `BROAGENTS\extension`.
5. Откройте вкладки ChatGPT и/или Gemini.
6. Дождитесь, пока вкладки появятся в интерфейсе как агенты.

Важно: интерфейс не должен сам создавать новые чаты под роли. Если открыто 1-2 чата, BROAGENTS работает с ними и позволяет просто переназначить роли.

## Что за что отвечает

- `server.js` держит HTTP API и WebSocket связь с расширением и интерфейсом.
- `dist/` это готовый веб-интерфейс, который сервер раздает сам.
- `extension/` подключает вкладки ChatGPT и Gemini к локальному серверу.
- `desktop-app/` это готовое отдельное приложение для запуска интерфейса как desktop app.
- `app-source/` это исходники интерфейса и desktop shell на случай правок и доработок.

## Какие роли использовать

- `Researcher` когда нужно быстро набросать варианты.
- `Critic` когда нужно проверить идею на риски и дыры.
- `Synthesizer` когда нужен один итог и один лучший следующий шаг.

## Рекомендуемый порядок

1. Сначала дайте задачу `Researcher`.
2. Потом передайте результат в `Critic`.
3. После критики передайте вывод в `Synthesizer`.

## Когда какой режим брать

- `fast` для черновиков и быстрых проходов.
- `checked` для обычной рабочей рутины.
- `full` для важных решений, клиентских материалов и чувствительных изменений.

## Правила живых тестов

- Не открывайте, не закрывайте и не перефокусируйте вкладки только ради теста.
- Не разворачивайте окно на весь экран и не сбрасывайте его состояние без прямой команды пользователя.
- Тестовые вопросы и короткие проверки пишите на русском и как обычный человек, а не как техническое задание.
- Для простых прогонов подходят фразы вроде `привет, как дела?`, `помоги коротко разобраться`, `собери краткий вывод`.
- При проверке режимов фиксируйте не только ответ, но и `requested mode`, `composerMode`, `busy` и последний ответ агента.
- Если текст похож на `??????` или `ÐŸÑ€Ð¸Ð²ÐµÑ‚`, его нельзя отправлять агенту. Сначала исправьте кодировку.

## Как безопасно слать русский текст из терминала

- Не используйте сырые inline-строки PowerShell для русских тестов, если нужна надежная передача без `????`.
- Не полагайтесь на PowerShell pipe для кириллицы.
- Используйте UTF-8 файл, который читает сам `node scripts/send-agent-message.js`.

```powershell
node .\scripts\send-agent-message.js --agent chatgpt-123 --file .\message.txt --wait
```

## Если что-то не работает

1. Проверьте `http://127.0.0.1:8080/health`.
2. Проверьте, что расширение загружено без ошибок.
3. Обновите вкладки ChatGPT и Gemini.
4. Убедитесь, что на порту `8080` не сидит чужой процесс.
5. Если открывается desktop app, а агентов нет, значит проблема не в приложении, а в расширении или вкладках.
